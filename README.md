# r2pq-sdk

R2PQ SDK is the official client library for interacting with the R2PQ suite.  
It supports PQ key generation, signing, verification, serialization, and transaction building.

## Purpose
- Developer-friendly high-level API  
- Unified PQ key + signature handling  
- Helpers for PQ transaction formatting  
- Integration with r2pq-cli and r2pq-devnet  

## Structure

/src              # Core SDK sources /types            # Shared interface definitions /examples         # Sample app + walkthroughs
